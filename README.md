# URL Shortener Web Application

A beginner-friendly URL Shortener built using Flask and SQLite.

## Features
- Shortens long URLs
- Stores URLs in database
- Redirects short URLs
- Displays URL history
- URL validation

## Tech Stack
- HTML
- Flask (Python)
- SQLite
- SQLAlchemy

## How to Run
1. Install dependencies  
2. Run `python app.py`  
3. Open http://127.0.0.1:5000/
